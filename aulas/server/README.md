# NLW - DIA 2 - BACK-END

## Criando projeto
Dentro da pasta desejada:
- Iniciar o projeto: 
    - `yarn init -y`
- Projeto com Typescript: 
    - `yarn add typescript -D`;
    - `yarn tsc --init`
        - Mudar a linha "target": "es5" <- para -> "target": "es2017" no arquivo tsconfig.json;

- Para que o node entenda o typescript:
    - `yarn add ts-node-dev -D`.
